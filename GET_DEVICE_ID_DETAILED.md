# 📱 Alexa App から Device ID を取得する - 超詳細ガイド

**このガイドは、スマートフォンを持っている方向けです。**
**iPhone（iOS）と Android の両方の手順を説明しています。**

---

## 前提条件

- ✅ スマートフォンを持っている
- ✅ Alexa App がインストール済み
- ✅ Alexa デバイス（Echo など）が自宅にある
- ✅ Amazonアカウントでログイン済み

---

## 📱 iPhone（iOS）の場合

### ステップ1: Alexa App を開く

1. **iPhone のホーム画面を見ます**

2. **Alexa App のアイコンを探します**
   - 青い背景に白い波線が描かれたアイコン
   - 見当たらない場合：
     - 画面下の「App Store」を開く
     - 検索タブで「Amazon Alexa」と入力
     - 「インストール」をタップ

3. **Alexa App をタップして開きます**

### ステップ2: Alexa App が起動したら

1. **画面の下に「≡」（三本線のメニュー）が見えます**
   - これは「ハンバーガーメニュー」と呼ばれています

2. **左下の「≡」をタップします**

   ```
   画面イメージ：
   ┌─────────────────────┐
   │                     │
   │   Alexa App の画面   │
   │                     │
   │                     │
   ├─────────────────────┤
   │ ≡    🏠   🔔   👤  │  ← 下部メニュー
   └─────────────────────┘
     ↑
     このボタンをタップ
   ```

### ステップ3: メニューが開きます

1. **「≡」をタップすると、左側からメニューが出てきます**

2. **メニューの中から「デバイス」を探します**

3. **「デバイス」をタップします**

   ```
   メニューイメージ：
   ┌──────────────────┐
   │ □ Alexa          │
   │ □ ルーティン      │
   │ □ デバイス ← ここをタップ
   │ □ スキル         │
   │ □ 設定           │
   │ □ ヘルプ         │
   └──────────────────┘
   ```

### ステップ4: デバイス一覧が表示されます

1. **「デバイス」をタップすると、自宅にあるすべての Alexa デバイス一覧が表示されます**

   例：
   ```
   デバイス一覧
   ┌────────────────────┐
   │ 📻 リビングのEcho  │
   │                    │
   │ 📱 寝室のEcho Show │
   │                    │
   │ 🔊 キッチンのEcho  │
   └────────────────────┘
   ```

2. **通知を受け取りたいデバイスをタップします**

   例：「リビングのEcho」なら、そこをタップ

### ステップ5: デバイスの詳細ページが開きます

1. **デバイスをタップすると、詳細情報が表示されます**

2. **画面を下にスクロール（下にスワイプ）して、詳細情報を探します**

3. **以下のような情報が表示されます：**

   ```
   デバイス情報
   ┌────────────────────────────────────┐
   │ リビングのEcho                     │
   │                                    │
   │ デバイス名: リビングのEcho         │
   │                                    │
   │ シリアル番号: G8E6Q2ABC1234        │
   │                                    │
   │ Device ID:                         │
   │ amzn1.ask.device.ABCDEFGHIJKL... │ ← これです！
   │                                    │
   │ WiFi: Alexa_Home_2.4GHz          │
   │                                    │
   │ ソフトウェアバージョン: 123.456   │
   └────────────────────────────────────┘
   ```

### ステップ6: Device ID をコピー

1. **「Device ID:」の隣にある長い文字列が「Device ID」です**

   例：
   ```
   amzn1.ask.device.ABCDEFGHIJKLMNOPQRSTUVWXYZ
   ```

2. **その文字列を長押しします**

3. **「コピー」が表示されたら、「コピー」をタップします**

   ```
   コピーメニュー
   ┌──────────────┐
   │ コピー        │ ← タップ
   │ 検索          │
   │ Web検索       │
   └──────────────┘
   ```

4. **Device ID がクリップボードにコピーされました** ✅

## 🎯 Device ID をコピーしたら

### テキストエディタに貼り付け

1. **メモアプリを開く**
   - iPhone：「メモ」アプリ
   - Android：「Google Keep」など

2. **以下のようにメモします**

   ```
   ========== Alexa認証情報 ==========
   
   Device ID:
   amzn1.ask.device.ABCDEFGHIJKLMNOPQRSTUVWXYZ
   
   （後で他の情報も追加します）
   ```

3. **このメモは後で使用するので、保存しておいてください**

## 🔍 Device ID の見分け方

### 正しい Device ID の例

```
amzn1.ask.device.ABCDEFGHIJKLMNOPQRSTUVWXYZ1234567890ABCDEFGH
```

### 特徴

- ✅ 「amzn1.ask.device.」で始まる
- ✅ その後ろに長いランダムな英数字が続く
- ✅ 合計50文字以上ある

### よく間違えられるもの

❌ **シリアル番号ではなく Device ID を使ってください**

```
シリアル番号（間違い）：
G8E6Q2ABC1234

Device ID（正しい）：
amzn1.ask.device.ABCDEFGHIJKLMNOPQRSTUVWXYZ...
```

## 🆘 Device ID が見つからない場合

### 原因 1: デバイス情報が表示されていない

**解決方法:**
- 画面を下にスクロール（下方向にスワイプ）してください
- デバイス詳細情報はページの下部にあります

### 原因 2: 複数のデバイスがある

**確認方法:**
- 複数のデバイスがある場合、それぞれ異なる Device ID を持っています
- **通知を受け取りたいデバイス** を選択してください
- 例：キッチンの Echo に通知したい → キッチンの Echo を選択

### 原因 3: Alexa App が最新版ではない

**解決方法:**
- App Store（iPhone）または Google Play（Android）で更新
- 「Amazon Alexa」を検索して「アップデート」をタップ

## 💡 複数のデバイスがある場合

**3台の Echo がある例：**

```
1. リビングの Echo
   Device ID: amzn1.ask.device.ABC123...

2. 寝室の Echo Show
   Device ID: amzn1.ask.device.DEF456...

3. キッチンの Echo
   Device ID: amzn1.ask.device.GHI789...
```

**どれを選べばいい？**

- 💬 毎日見ているデバイス
- 📍 リビングなど、人が頻繁に集まる場所
- 📺 Echo Show など、画面が大きいデバイス

**→ おすすめはリビングの Echo です**
